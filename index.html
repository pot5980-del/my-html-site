자주 하는 질문 (FAQ)

시스템 구성, 기본 설정, 라이브러리 및 API 사용법 등 자주 접수되는 기본 문의사항입니다.

  • .NET Framework 및 라이브러리 필수 설치 안내
  • i-MATRIX 세션 변수와 전역 변수 사용법
  • 로우코드 기반 AUD플랫폼 아키텍처 구성

신규 기술 및 업데이트

새롭게 추가된 플랫폼 기능 설명, 패치 내역 및 활용도 높은 최신 등록 기술 자료입니다.

  • [i-AUD] 누적형 차트 데이터 레이블 표현 기능
  • (BCRM#D0453) Portal Q&A 게시판 (사용자 게시판)
  • [i-META] 값 목록(LOV) 조회조건 확인 방법 개선

분석 및 화면 개발 (i-AUD/i-MATRIX)

다차원 분석(OLAP), 차트, 그리드 컴포넌트 활용 및 정형 보고서 작성 가이드입니다.

  • [i-AUD] GeoMap 컴포넌트 (지도, 맵) 사용 가이드
  • [i-AUD] i-OLAP 기본 연산 방식 및 옵션 설정
  • [i-MATRIX] 화면 개발 시 유의사항 및 API 가이드

서버 운영 및 메타데이터 (Server/i-META)

AUD Server 환경 구성, 시스템 프로퍼티 설정 및 i-META/i-PORTAL 메타데이터 관리 가이드입니다.

  • AUD Server 구동 환경 변수 및 프로퍼티 설정
  • [i-META] 데이터 소스 활용 및 권한 제어 가이드
  • [i-PORTAL] 사용자 권한 관리 및 게시판 설정
